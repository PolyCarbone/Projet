# Guide d'utilisation des bannières

## Emplacement des images

Placez vos images de bannières dans ce dossier avec les noms suivants :
- `forest.jpg` - Bannière thème forêt
- `ocean.jpg` - Bannière thème océan
- `mountain.jpg` - Bannière thème montagne
- `sunset.jpg` - Bannière thème coucher de soleil
- `space.jpg` - Bannière thème espace

## Format recommandé

- Format : JPG ou PNG
- Dimensions : 1200x400 pixels (ratio 3:1)
- Taille de fichier : < 1 MB par image
- Style : Photos écologiques et naturelles

## Sources d'images gratuites

- Unsplash : https://unsplash.com/
- Pexels : https://pexels.com/
- Pixabay : https://pixabay.com/
